Es el PAGO del CRÉDITO que otorgo el PROVEEDOR de SERVICIO o PRODUCTO OPERATIVO (Gasto Operativo).
Si el PAGO proviene de una Factura, escribe el importe del IVA (donde señale la flecha). El IVA del gasto servira para deducir ante el SAT (IVA Accreditable).
La FORMA DE PAGO al Proveeedor (Acreedor) puede ser:
1) EFECTIVO
2) BANCO (Cheque, transferencia o tarjeta de débito)
